# Regex Helper — 正则表达式助手

## 功能说明

辅助正则表达式的编写和调试。

## 使用方法

### 1. 编写正则

```
用户: 写一个匹配邮箱地址的正则表达式
```

### 2. 测试正则

```
用户: 测试这个正则是否能匹配：
Pattern: \d{4}-\d{2}-\d{2}
Text: 2026-04-06
```

### 3. 解释正则

```
用户: 解释这个正则的含义：
^(?=.*[A-Z])(?=.*[a-z])(?=.*\d)[A-Za-z\d]{8,}$
```

### 4. 常用正则库

- 邮箱
- 手机号
- URL
- IP地址
- 日期
- 身份证
